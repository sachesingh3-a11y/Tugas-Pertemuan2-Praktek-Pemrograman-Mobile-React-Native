# 🪪 Kartu Identitas Digital Mahasiswa

Aplikasi mobile sederhana untuk menampilkan identitas diri dan motivasi pribadi, dibangun dengan teknologi React Native dan Expo.

## 📸 Screenshots
![Preview Aplikasi](assets/images/screenshot.png)

## 🛠️ Tech Stack
- **Framework:** React Native
- **Runtime:** Expo (SDK 50)
- **Navigation:** Expo Router
- **Styling:** StyleSheet (Native)
- **Language:** TypeScript

## 🚀 Fitur Utama
- **Identity Display:** Menampilkan Foto, Nama, NIM, dan Jurusan secara clean.
- **Modern UI:** Menggunakan *shadow* dan *border-radius* untuk tampilan kartu yang estetik.
- **Responsive Layout:** Tata letak yang dioptimalkan untuk berbagai ukuran layar mobile.
